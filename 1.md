## ข้อที่ 1

คำถาม: คุณต้องการรายงานกิจกรรมที่ผิดปกติจากโดเมนหนึ่งใน netacad.com คุณจะติดต่อพวกเขาได้อย่างไร 

จำนวนครั้งสูงสุดที่ลองได้: 5 ครั้ง 

คำตอบนี้ต้องตรงตามตัวพิมพ์ใหญ่เล็ก

คำตอบควรอยู่ในรูปแบบxxxxxxx@xxxxx.xxx

โดยที่ x คือตัวอักษร

คำตอบ: infosec@cisco.com

# Step 1

คำตอบต้องอยู่ในรูปแบบ Email เราจึงใช้คำสั่งนี้
```
whois netacad.com | grep -i "email"
```
### อธิบายคำสั่ง

1. whois เป็นเครื่องมือสำหรับสืบค้นข้อมูลสาธารณะของชื่อโดเมนและ IP Address ช่วยให้เราตรวจสอบได้ว่าใครเป็นเจ้าของโดเมน วันที่จดทะเบียน วันหมดอายุ ผู้ให้บริการรับจดทะเบียน และข้อมูล Name Server
2. netacad.com คือโดเมนเป้าหมายของเรา
3. | อันนี้เป็นเครื่องหมายที่ทำหน้าที่ส่งต่อผลลัพธ์ (Output) ของคำสั่งทางซ้ายมือ ไปเป็นข้อมูลนำเข้า (Input) ของคำสั่งทางขวามือ
4. grep ทำหน้าที่ค้นหาข้อความหรือคำที่ต้องการภายในไฟล์ หรือผลลัพธ์จากคำสั่งอื่นๆ ผ่านการใช้ Pipe (|) แล้วแสดงผลเฉพาะบรรทัดที่พบข้อมูลนั้นออกมา
   - -i เป็นการสั่งให้ระบบไม่ต้องสนใจตัวอักษรพิมพ์เล็กหรือพิมพ์ใหญ่ ทำให้ไม่ว่าจะเขียนคำที่ต้องการหาด้วยรูปแบบใดก็ตาม เช่น Email, email, หรือ EMAIL คำสั่ง grep ก็จะสามารถตรวจจับและดึงบรรทัดนั้นออกมาแสดงบนหน้าจอได้
5. "email" คือคำที่เราสั่งให้ค้นหา
